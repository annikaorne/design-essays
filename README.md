# Design Essays

A collection of essays on the principles and practice of design, by Annika Orne.

## Structure

- `index.html` — landing page
- `essays/` — individual essays, each in its own folder with an `index.html` and `images/`
  - `honesty-in-design/` — *Honesty as a Design Principle* (April 2026)

## Adding a new essay

1. Create a new folder under `essays/` (e.g., `essays/next-essay/`)
2. Add `index.html` and an `images/` folder inside
3. Add a new `<li>` to the essays list in the root `index.html`
